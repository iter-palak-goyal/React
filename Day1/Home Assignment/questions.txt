Q1. Difference between Library and Framework
Library                                         |            Framework
A collection of functions you call when needed	| A structure that calls your code
You control the flow	                        | Framework controls the flow
Flexible	                                    | Opinionated
Used to solve specific tasks	                | Used to build complete apps
Library: React, Lodash                          |Framework: Angular, Django, Spring
In library we call the code, in framework the code calls us (Inversion of Control).


Q2. Single Page App vs Multi Page App
SPA                                        |                     MPA
One HTML page                              |             Multiple HTML pages
Uses client-side routing                   |             Uses server-side routing
Fast after first load	                   |             Slow after first loader navigation
Better UX (no reload)                      |    Page reload every click
Example: Gmail, Facebook	               |    Example: Amazon, traditional sites

O8. React Reconciliation Process
Reconciliation =Process of updating the DOM efficiently.
Steps:
State changes
New Virtual DOM created
Compare with old VDOM
Find differences
Update only changed parts in real DOM
Goal → Minimum DOM operations


Q9Virtual DOM (VDOM)
Lightweight copy of real DOM (JavaScript object).
Why needed?
Real DOM updates are slow.
Process:State change → VDOM update → Diff → Real DOM update
Benefits:
Fast rendering
Better performance
Less reflow/repaint


O10. Diffing Algorithm
React compares old vs new VDOM.
Rules React assumes:
Different element type → replace entire node
Same type → update attributes only
Lists use keys to track items
Example:
<li key={id}>Item</li>
Keys help React identify:
Added
Removed
Reordered items


Q11. Rules of JSX
Rule 1: Return single parent element
Rule 2: HTML in lowercase
Rule 3: Close all tags
Rule 4: Custom Tags should be in CamelCase 
Rule 5: JS inside {}
Rule 6: We should not use the JS Keywords in the JSX
